YRM2016

Registration is now closed, however if you wish to join the waiting list in case of cancellations, please follow this link

The fee for the conference is:

  1. 100£ without accommodation,
  2. 200£ with accommodation (private room, the number of these is limited),
  3. 180£ with accommodation (shared room).

The rooms are booked from Sunday, 5th of June* since the conference starts at 9am the following day.

Here you can find link for paying the conference fees.
Please register on the page, choose the suitable accommodation option and make the transaction using a credit/debit card.
Keep in mind that the number of private rooms is limited.
The deadline for the payment is 11/05/16.

*If there is any doubt, "from Sunday" does not include the preceding Saturday and Friday.
